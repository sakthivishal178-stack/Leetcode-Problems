Approach

The goal is to find the maximum number of consecutive 1s in the array.

We keep a pointer left to mark the start of the current sequence of 1s.

Traverse the array using the right pointer.
Whenever we encounter a 0, the current sequence of 1s ends.
Calculate its length using right - left and update the maximum length.
Start a new sequence by setting left = right + 1.
After the loop, check the last sequence because the array may end with 1s.
Complexity

Time Complexity: O(n)

We traverse the array only once.

Space Complexity: O(1)

Only a few variables are used.

Code
class Solution:
    def findMaxConsecutiveOnes(self, nums: List[int]) -> int:
        left = 0
        cons = 0

        for right in range(len(nums)):
            if nums[right] != 1:
                cons = max(cons, right - left)
                left = right + 1

        cons = max(cons, len(nums) - left)

        return cons
