Approach

We need to find the longest subarray containing at most k zeros, since we can flip at most k zeros into ones.

We use a sliding window:

Initialize two pointers, left and right, to represent the current window.
Move the right pointer through the array.
Whenever we encounter a 0, increment the zero counter.
If the number of zeros exceeds k, shrink the window from the left until it becomes valid again.
After each valid window, update the maximum window length.
Return the maximum length found.
Complexity

Time Complexity: O(n)

Each element is visited at most twice (once by right and once by left).

Space Complexity: O(1)

Only a few variables are used regardless of the input size.

Code
class Solution:
    def longestOnes(self, nums: List[int], k: int) -> int:
        left = 0
        zero = 0
        max1 = 0

        for right in range(len(nums)):
            if nums[right] == 0:
                zero += 1

            while zero > k:
                if nums[left] == 0:
                    zero -= 1
                left += 1

            max1 = max(max1, right - left + 1)

        return max1
