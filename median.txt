Approach
Merge nums1 and nums2 into a new array arr.
Sort arr using Selection Sort:
For each position, find the smallest element in the remaining part of the array.
Swap it into its correct position.
Check the length of the sorted array:
If odd, return the middle element.
If even, return the average of the two middle elements.
Complexity
Time Complexity:
Selection Sort takes O(n²) time, where n = len(nums1) + len(nums2).
Space Complexity:
O(n) because a new merged array is created.
Code
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        arr = nums1 + nums2
        n = len(arr)

        for i in range(n):
            position = i

            for j in range(i + 1, n):
                if arr[j] < arr[position]:
                    position = j

            arr[position], arr[i] = arr[i], arr[position]

        if len(arr) % 2 == 0:
            return float(
                (arr[(len(arr) // 2) - 1] + arr[len(arr) // 2]) / 2
            )
        else:
            return float(arr[len(arr) // 2])
