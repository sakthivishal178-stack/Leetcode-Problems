Intuition

To reach any stair i, we can only come from either stair i-1 or stair i-2. Since we want the minimum total cost, we store the minimum cost required to reach each stair and build the answer from previously computed values.

Approach
Let dp[i] represent the minimum cost required to reach the i-th stair.
Initialize:
dp[0] = cost[0]
dp[1] = cost[1]

For every stair from 2 to n-1, compute:

dp[i] = cost[i] + min(dp[i-1], dp[i-2])
The top is beyond the last stair, so the final answer is the minimum cost of reaching either of the last two stairs.
Complexity
Time Complexity: O(n)
Space Complexity: O(n)
Code
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        dp = [0] * len(cost)

        dp[0] = cost[0]
        dp[1] = cost[1]

        for i in range(2, len(cost)):
            dp[i] = cost[i] + min(dp[i - 1], dp[i - 2])

        return min(dp[-1], dp[-2])
